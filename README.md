## 操作方法

行为|功能
:---|:---
单击|选中/拖动
双击|创建新点/打开属性窗口
按住 <kbd>Ctrl</kbd>|不自动对齐到其他点
按住右键|拖动画布
<kbd>Delete</kbd>|删除
<q>Save</q>按钮|写入数据库
